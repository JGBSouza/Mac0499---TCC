# Automatização do fluxo de submissões de patches em contribuições do Kernel Linux através do software livre kworkflow

Este Trabalho de Conclusão de Curso tem por objetivo contribuir com o software livre kworkflow, automatizando o fluxo de submissão de patches para contribuidores do kernel linux, permitindo maior visibilidade e controle dos patches submetidos, bem como um ambiente independente, acessível através do terminal por linha de comando.
